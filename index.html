Tentang Saya

Perkenalkan saya Muhammad Riduan adalah seorang junior front end web developer yang sedang menempuh pendidikan D4 di Politeknik Negeri Samarinda. Meskipun masih dalam tahap awal karier, Saya telah menunjukkan bakat dan minat yang besar dalam pengembangan web. Sebagai seorang junior,saya terus belajar dan memperluas pengetahuannya dalam bidang front end development. Dalam perann sebagai junior front end web developer,Saya bertanggung jawab untuk mengimplementasikan desain web yang telah dibuat oleh desainer atau tim desain.Saya bekerja dengan bahasa pemrograman seperti HTML, CSS, JavaScript Serta Framework Codeigniter dan laravel untuk menciptakan tampilan yang menarik dan interaktif untuk situs web. Saya memiliki pemahaman yang baik tentang struktur HTML untuk menyusun konten situs dengan baik, serta kemampuan dalam CSS untuk merancang tata letak dan gaya visual yang konsisten dan menarik. Meskipun masih dalam tahap awal karier, Saya telah mulai mengembangkan keterampilan dalam penggunaan framework front end seperti Bootstrap untuk mempercepat proses pengembangan dan memastikan responsivitas situs web pada berbagai perangkat dan ukuran layar. Saya juga memiliki minat dalam mempelajari dan menerapkan teknologi baru dalam pengembangan web. Selain itu, Saya juga memiliki keinginan untuk memahami prinsip-prinsip desain antarmuka pengguna (UI) dan pengalaman pengguna (UX) untuk memastikan situs web yang dikembangkannya tidak hanya tampak bagus secara visual, tetapi juga mudah digunakan dan menghasilkan pengalaman yang menyenangkan bagi pengguna. Dengan semangat belajar yang tinggi dan dedikasi terhadap pengembangan web, saya siap untuk menghadapi tantangan baru dalam perannya sebagai junior front end web developer. Saya berharap untuk terus berkembang dan mengeksplorasi potensinya dalam bidang ini, serta memberikan kontribusi yang berarti dalam pengembangan proyek-proyek web yang inovatif dan bermutu tinggi.
